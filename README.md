1. 将复制粘贴数据都改为深拷贝，提升用户体验
2. 全局监听键盘事件改为仅在编辑器监听键盘事件

## 9月4日更新内容：
1. 为了提高效率，取消删除的弹窗提示

4. 同时修复了偶尔不能导出html文件的bug
